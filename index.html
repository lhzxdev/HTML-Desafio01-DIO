Client x Server

(inicio)

O tema Cliente e Servidor é de total importância para todos os programadores Web. Entender a relação entre os dois é de extrema importância e algo que vai ser levado para o restante da vida como desenvolvedor.

O client Side, são todas as tecnologias que rodam do lado do cliente, também podemos chamar de usuário final. Dois pontos importantes aqui, é o UI Design, ou seja, a aparência que o usuário vai ter ao abrir uma aplicação web e também o UX Design, a experiência que o usuário vai ter ao utilizar o site.

Já o Server Side, é tudo que o usuário não vê, mas está ali respondendo solicitações a todo momento, entregando requisições das tecnologias utilizadas e buscando itens armazenados no banco de dados.

O que é HTML?

(inicio)

HTML é uma linguagem de marcação de hipertexto e compõe boa parte das páginas webs que visitamos a todo momento. O HTML não é considerado uma linguagem de programação, mas tem sua grande importância quando o quesito é criação de aplicações web.

Criado por Tim Berners-Lee em 1991 como base da World Wide Web, a linguagem de marcação evoluiu, permitindo a adição de imagens, formulários e estilo, esse ultimo utilizando o CSS, que é uma linguagem própria para estilização das páginas.

Tags em HTML

(inicio)

O HTML funciona através da declaração de TAGS e diversos atributos que podemos utilizar para dar forma e contexto a um site. Essas Tags ajudam a elaborar a semântica na formatação do HTML e gerar conteúdos de forma organizada. As principais Tags iniciais para o Html, são: html que abre e fecha o documento, fazendo o navegador entender que se trata de um doc HTML. A Tag head, que contem itens a serem pré carregados antes de todo conteúdo e a tag body, que contém todo o corpo do html, ou seja, aquilo que o usuário realmete ver.

Tags H1 - H6

(inicio)

As tags de cabeçalho em HTML são usadas para definir os títulos e subtítulos em uma página da web. Elas são essenciais para estruturar o conteúdo e fornecer uma hierarquia clara para os motores de busca e leitores de tela. Existem seis níveis de cabeçalhos, de H1 a H6, cada um representando uma importância decrescente.

Aqui está uma visão geral de cada uma das tags:

h1: É o cabeçalho mais importante e deve ser usado para o título principal da página. Geralmente, há apenas um h1 por página, representando o tema central do conteúdo.
h2: Usado para títulos secundários que se relacionam diretamente com o conteúdo do h1. Eles podem ser considerados subseções importantes.
h3: Indicado para títulos de nível 3, normalmente relacionados a subseções dentro de um h2.
h4: Representa um cabeçalho de nível 4, geralmente usado para títulos menos relevantes em relação aos h3.
h5: Utilizado para títulos de nível 5, subordinados aos h4.
h6: O cabeçalho menos importante, normalmente usado para títulos menos relevantes dentro de um contexto específico.

Tag A

(inicio)

A tag a em HTML é usada para criar links, permitindo que os usuários naveguem para outras páginas da web, arquivos, seções da mesma página ou até mesmo endereços de e-mail. Ela é um dos elementos mais importantes para a interconexão entre páginas e recursos na internet

Para usar a Tag A, é necessario passar alguns atributos, que são eles:
Href: O atributo obrigatório que especifica o destino do link. Pode ser uma URL, um caminho relativo ou uma âncora interna.
Texto do link: O conteúdo visível do link, ou seja, o texto que será clicável.

Além do atributo href, a tag a pode ser usada em conjunto com outros recursos, como o atributo target, que define como o link deve ser aberto:
_self: O link abre na mesma janela/tab que o link foi clicado (padrão).
_blank: O link abre em uma nova janela/tab do navegador.
_parent: O link abre no frame pai (se a página estiver em um frame).
_top: O link abre na janela completa do navegador, substituindo todos os frames (se a página estiver em um frame).

Listas com HTML

(inicio)

Em HTML, você pode criar três tipos diferentes de listas: listas ordenadas (ol), listas não ordenadas (ul) e listas de descrição (dl). As listas são usadas para organizar e estruturar informações de forma clara e concisa.
Listas Ordenadas (ol): As listas ordenadas são usadas para listar itens em uma sequência específica, geralmente numerada ou com marcadores. Cada item da lista é criado usando a tag li (item de lista).
Listas Não Ordenadas (ul): As listas não ordenadas são usadas para listar itens sem uma sequência específica. Normalmente, são usados marcadores, como pontos ou círculos, para cada item da lista.
Listas de Descrição (dl): As listas de descrição são compostas por pares de termos (dt) e suas descrições (dd). Elas são frequentemente usadas para criar glossários ou explicações de termos.

Citações com HTML

(inicio)

A tag blockquote em HTML é usada para destacar um trecho de texto que é uma citação de outra fonte. É comumente usada para mostrar citações ou passagens significativas de um autor ou de outra fonte em um bloco de texto separado. O conteúdo dentro da tag blockquote é tipicamente exibido com uma formatação especial para indicar que é uma citação.
